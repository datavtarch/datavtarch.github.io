## Render video
Create a local MP4 from the website with Chrome and FFmpeg:

```bash
npm run video:home -- --duration 10 --fps 30 --width 1080 --height 1920 --out exports/vtarch-home.mp4
```

Useful presets:

```bash
npm run video:home -- --width 1080 --height 1920 --out exports/reels.mp4
npm run video:home -- --width 1920 --height 1080 --out exports/youtube.mp4
```
